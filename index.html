Marché UGC
Achète et revends tes items Roblox
▸
1
Connecte-toi
Entre ton pseudo Roblox et un mot de passe pour accéder au hub. Ton compte est sauvegardé localement.
2
Clique sur "+ Vendre un UGC"
Remplis le nom de ton UGC, la catégorie, le prix en Robux, la rareté et un emoji pour le représenter.
3
Publie ton annonce
Ton UGC apparaît dans le marché. Les autres membres peuvent te contacter via Discord pour acheter.
4
Finalise la vente sur Roblox
Utilise toujours le système officiel de Roblox pour transférer les items. Ne partage jamais ton mot de passe Roblox.
💡 Ajoute une description claire et un prix juste pour vendre plus vite !

Générateur GFX IA
Crée des visuels Roblox avec l'IA
▸
1
Obtiens une clé Groq gratuite
Va sur console.groq.com → crée un compte gratuit → API Keys → Create API Key. Copie la clé (commence par gsk_).
2
Choisis un style
Sélectionne parmi Action, Cinéma, Dark, Réaliste, Anime ou Fantasy selon le rendu que tu veux.
3
Décris ta scène
Écris ce que tu veux voir dans ton GFX. Ex: "guerrier avec une épée de feu debout sur un toit la nuit".
4
Génère et utilise le prompt
Clique sur Générer, copie le prompt et colle-le dans Leonardo.ai ou Bing Image Creator pour avoir ton image.
🔗 Leonardo.ai est gratuit et donne les meilleurs résultats pour les GFX Roblox !

Enchères Live
Vends tes UGC au plus offrant en temps réel
▸
1
Lance ton enchère
Clique sur + Créer une enchère dans l'onglet Enchères. Entre l'ID ou l'URL de ton UGC — les infos se remplissent automatiquement.
2
Configure les paramètres
Fixe un prix de départ, une mise minimum par offre, et une durée (5 min à 24h). Ajoute ton lien Game Pass Roblox si tu en as déjà un.
3
Les membres enchérissent
Tout le monde voit l'enchère en direct. Le timer décompte en temps réel. Les offres montent automatiquement. Le badge devient rouge dans les 60 dernières secondes !
4
Fin de l'enchère → paiement
Quand le timer atteint 0, le gagnant voit un bouton Acheter le Game Pass. Il clique → achète le Game Pass sur Roblox → les Robux arrivent sur ton compte (moins 30% de commission Roblox).
5
Livraison de l'UGC
Une fois le paiement confirmé, envoie l'UGC au gagnant via le trade officiel Roblox. Contacte-toi sur Discord pour coordonner la livraison.
💡 Conseils pour bien vendre
• Mets un prix de départ bas pour attirer plus d'enchérisseurs
• Choisis une durée courte (5-10 min) pour créer de l'urgence
• Crée ton Game Pass avant de lancer l'enchère pour gagner du temps
• Roblox prend 30% de commission sur chaque Game Pass vendu
• Ne livre jamais l'UGC avant d'avoir reçu les Robux
⚠️ Sécurité
Ne donne jamais ton UGC avant le paiement Game Pass. En cas de problème avec un acheteur, signale-le sur le serveur Discord RealBlue.
FAQ
Questions fréquentes
Est-ce que Vertex Hub est officiel Roblox ?▸
Non. Vertex Hub est une plateforme communautaire indépendante. Nous ne sommes pas affiliés à Roblox Corporation. Utilise toujours les systèmes officiels de Roblox pour les transactions.
Mon mot de passe Vertex Hub est-il lié à Roblox ?▸
Non, ce sont deux choses différentes. Ton mot de passe Vertex Hub est uniquement pour te connecter au hub. Ne rentre JAMAIS ton vrai mot de passe Roblox ici.
La clé Groq pour le GFX est-elle vraiment gratuite ?▸
Oui ! Groq offre un plan gratuit très généreux. Rends-toi sur console.groq.com, crée un compte et génère une clé API. Tu peux générer des centaines de prompts sans payer.
Comment sécuriser mes échanges UGC ?▸
Utilise uniquement le système de trade officiel de Roblox. Vérifie le profil Roblox de l'autre personne. Ne fais jamais un échange hors plateforme Roblox et ne donne jamais tes informations personnelles.
Mes données sont-elles sauvegardées ?▸
Toutes les données (annonces, échanges, compte) sont sauvegardées localement dans ton navigateur via localStorage. Elles restent sur ton appareil et ne sont pas envoyées sur un serveur.
Comment créer un Game Pass pour mon enchère ?▸
Va sur roblox.com → Créer → Game Passes → Nouveau Game Pass. Mets le nom de ton UGC et le prix final de l'enchère. Roblox prend 30% donc si tu veux recevoir 700 R$, mets le prix à 1000 R$. Une fois créé, copie le lien et envoie-le au gagnant.
